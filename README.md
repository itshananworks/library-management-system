# 📚 Library Management System (Python)

## 🧠 Project Overview

This is a simple console-based Library Management System built using Python.
The project helps practice fundamental programming concepts such as lists, loops, conditionals, and user input handling.

---

## 🚀 Features (Version 1)

* View all books in the library
* Add new books
* Delete books by number
* Exit the program safely
* Input validation for menu selection
* Handles empty library cases

---

## 🛠️ Technologies Used

* Python 3
* Basic data structures (List)
* Control flow (if/elif/else)
* Loops (while, for)
* Built-in functions (len, input, print)

---

## 📦 How to Run

```bash
python library.py
```
